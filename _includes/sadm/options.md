## OPTIONS

| Options           | Description                                |
|:-------:          |:------------------------------------------ |
| **[ -d 0-9 ]**    | Set debug level from 0 to 9 (Default is 0) |
| **[ -h ]**        | Display this help and exit.                |
| **[ -v ]**        | Output version information and exit.       |
